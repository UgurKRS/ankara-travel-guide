Atribuciones de Herramientas y API

Este proyecto utiliza varias herramientas y servicios de terceros para su funcionalidad:

  • OpenWeatherMap API: Los datos meteorológicos en vivo que se muestran en el widget de la página de inicio son proporcionados por OpenWeatherMap.
  • basicLightbox (NPM): La funcionalidad de "lightbox" (popup de imágenes) en las páginas de detalle es posible gracias a la biblioteca basicLightbox, instalada vía npm.
  • JavaScript (Fecha Automática): Se ha utilizado JavaScript "vanilla" para obtener la fecha del año actual y mostrarla dinámicamente en el pie de página (footer).
  • Parcel: El empaquetador (bundler) de código utilizado para compilar el sitio, optimizar los archivos y ejecutar el servidor de desarrollo. https://parceljs.org/
  • PostHTML-Include: El pre-procesador de HTML (instalado vía npm) que nos permitió usar <include> para los parciales (header y footer). https://github.com/posthtml/posthtml-include
  • Google Fonts: Las tipografías del sitio (Playfair Display y Roboto) se sirven a través de Google Fonts.

Declaración de Uso de IA Generativa

Para la obtención de contenido, se ha utilizado una herramienta de IA generativa (Google Gemini) para ayudar en la redacción de algunos textos descriptivos.

El uso de esta herramienta se limitó a:

  • Generar los párrafos de bienvenida y descripción de la página de inicio.
  • Redactar las descripciones detalladas para las páginas de atracciones (como Anıtkabir y el Castillo de Ankara).
  • Redactar los resúmenes de las tarjetas de atracciones en la página de "Lugares para Visitar".
